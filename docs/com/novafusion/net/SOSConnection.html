Classpublic class SOSConnection
InheritanceSOSConnection flash.events.EventDispatcher

A class to help you use SOS max for your tracing quicker. This class has a number of methods that use predefined message keys in SOS max. There are also methods to set your own keys, send folded messages and so on.
